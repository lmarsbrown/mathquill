MathQuill Matrix Test

Type these commands:

  • \mat - matrix (no delimiters)
  • \pmat - parentheses matrix ( )
  • \bmat - bracket matrix [ ]
  • \vmat - vertical bar matrix | |

Navigation:

  • Left / Right - Move between columns
  • Up / Down - Move between rows
  • , (comma) - Add a new column
  • Enter - Add a new row
  • Backspace on empty cell in top row - Delete column
  • Backspace on empty cell in first column - Delete row
  • Backspace on empty cell otherwise - Move to left cell
